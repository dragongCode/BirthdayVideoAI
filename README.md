## Project Overview

- **Template-Driven Creation:** Generate personalized birthday videos based on handpicked, high-quality templates.
- **AI-Assisted Production:** Support AI-powered generation for scripts and visual content.
- **Core User Features:** Support user accounts, credits/points system, payments, and media asset uploads.
- **Modern Tech Stack:** Built with Next.js, Better Auth, Drizzle ORM, and Creem.
